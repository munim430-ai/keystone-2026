# Patterns & Techniques

## The Two-Gate Pipeline
**When to use**: tracking any student's progress.
**How**: gate 1 = university (documents → fee → submission → interview → admission → tuition+SWIFT → CoA); gate 2 = embassy (cover letter + affidavits + TB test → visa interview → approval).
**Trade-offs**: preparing gate-2 documents early costs stamp/notary fees, but bank statements and affidavits date-stamp — sequence them late; academics and FRC early.

## Translate-then-Notarize Chain
**When to use**: every Bangla-language identity/financial document in the DHL set.
**How**: color copy of original → certified translation → notary. Never reverse the order.
**Trade-offs**: batch-processing all documents in one notary visit saves cost, but if one document changes (new passport, corrected NID) its whole chain redoes.

## The Weakness-Reframe (SOP)
**When to use**: any weak grade, gap year, or low IELTS in the profile.
**How**: name the weakness → give the honest cause → show the corrective action → land on the measurable improvement. Never omit; silence reads as concealment.
**Trade-offs**: uses precious SOP space, but an unexplained gap invites interview probing you don't control.

## One-Story-Three-Places (Consistency Discipline)
**When to use**: always — before any submission.
**How**: gap years, sponsor identity, funding source, and return plan must be identical in SOP, cover letter, affidavits, and rehearsed interview answers. Diff-check the four before filing.
**Trade-offs**: none. Divergence is the most common self-inflicted rejection.

## Sponsor Name-Match Audit
**When to use**: assembling any financial bundle (mandatory for Gachon, wise everywhere).
**How**: sponsor's name spelled identically on bank statement, solvency, affidavit, TIN, trade license, and cover letter; verify character-by-character against passport/NID.
**Trade-offs**: fixing a bank-record spelling takes days — audit at intake, not at submission.

## Scholarship-Ladder Arithmetic
**When to use**: pitching cost to a family; choosing between universities.
**How**: compute total program cost = (tuition × semesters) − (scholarship % × applicable semesters). Compare shapes: 1-semester ladders (Kyungsung/Sejong/Dongshin/Gachon) vs multi-semester (aSSIST 3-sem 50%; Kyungsung masters 40% guaranteed at GPA 3.0+).
**Trade-offs**: an IELTS retake costing one band step can outperform months of haggling — e.g. Dongshin 6.0→6.5 moves 40%→50%.

## Affidavit 3-in-1 Assembly
**When to use**: preparing the sponsor's financial evidence.
**How**: one notarized instrument with sponsorship clauses + movable-property table + immovable-property table (full deed metadata) + annual-income table; USD conversion line ("1 USD = X BDT, dated, bb.org.bd") under every table; tuition-deposit clause if already paid; return undertaking last.
**Trade-offs**: collecting Khatian/Dag/Mouza deed data is slow — start it the day the family commits.

## The Known-Question Bank Drill
**When to use**: from admission-interview scheduling until embassy interview.
**How**: rehearse the 23 questions; select subject-set by major (CSE vs business); run the mandatory mock; enforce display-name/dress/background/connection protocol.
**Trade-offs**: over-rehearsal produces recited answers — drill content, not scripts.
